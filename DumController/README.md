# MakerLabVN: DumController

![](/image/ca2.jpg)

## Thông số kỹ thuật

### Mạch TX

- Điện áp sử dụng: 3VDC (2x pin AA)

### Mạch RX

- Điện áp sử dụng: 5~6VDC (4x pin AA)
- Dòng điện tối đa trên mỗi động cơ: 800mA

> **Lưu ý: Chỉ có port nguồn cấp là input, 3 port động cơ còn lại là output**
